In this page we list all the Workflow Engines that are supported by the SHIWA Repository.

If this image is displayed then it means that the associated Workflow Engine is not supported by the Submission Service. The decision to allow a Workflow Engine to be used by the Submission Service rests with the developer.

If this image is displayed then it means that the associated Workflow Engine is supported by the Submission Service. This implies that any Workflow Implementation that is associated to an Engine of this type can be flagged as Submittable by the developer. A developer can only flag their Workflow Implementation as Submittable if:

  • The Implementation of the Workflow and the Workflow itself must be flagged as public.
  • The Implementation must have a configured Submission Execution Node associated with it.
